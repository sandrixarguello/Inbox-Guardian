# Product Vision

## Mission

Inbox Guardian aims to help CRM and Email Marketing teams detect HTML issues before launching an email campaign.

The goal is to improve:

- Deliverability
- Accessibility
- HTML quality
- Email client compatibility

---

## Long-term Vision

Inbox Guardian will evolve into a complete Email QA platform capable of:

- HTML analysis
- Image validation
- Link validation
- Accessibility checks
- QA Score calculation
- Professional PDF reports
- Command Line Interface (CLI)
- Integration with Email Marketing platforms

---

## Target Users

- CRM Analysts
- Email Marketing Specialists
- Marketing Automation Engineers
- QA Engineers
- Developers

---

## Core Principles

- Simple
- Modular
- Extensible
- Well tested
- Open Source
