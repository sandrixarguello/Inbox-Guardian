# Email Deliverability Notes

Inbox Guardian includes several validations inspired by email marketing best practices.

---

## Images

Recommended:

- ALT attribute
- Width and height defined
- Optimized image size

---

## Links

Recommended:

- HTTPS
- Valid href
- UTM parameters for campaign tracking

Avoid:

- HTTP links
- Empty links

---

## HTML

Recommended:

- Table-based layouts
- Simple HTML structure
- Inline CSS

---

## Accessibility

Recommended:

- Images with ALT
- Buttons with readable text
- Proper heading structure
- Accessible tables

---

Future versions may include additional validations based on industry best practices.
