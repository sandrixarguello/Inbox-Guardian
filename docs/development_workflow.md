# Definition of Done

Una funcionalidad se considera terminada cuando:

- [ ] El código está implementado.
- [ ] Tiene tests automatizados.
- [ ] Todos los tests pasan (`pytest`).
- [ ] La documentación fue actualizada.
- [ ] El roadmap refleja el avance.
- [ ] El CHANGELOG fue actualizado.
- [ ] Se realizó un commit con un mensaje descriptivo.
- [ ] El Issue correspondiente fue cerrado.
