# 📬 Inbox Guardian

Inbox Guardian is an Email Deliverability & Quality Analyzer built in Python.

## Project Status

🚧 In Development

## Goals

- Analyze HTML emails
- Calculate text/image ratio
- Validate deliverability best practices
- Generate QA reports

## Tech Stack

- Python
- BeautifulSoup
- Streamlit
- Pandas
